# ConL-IME

## 支持的操作系统

ConL-IME 支持主流的桌面和移动操作系统。

+ [Desktop Operating System] (桌面平台操作系统)
    + [X] [Windows] 7/8/8.1/10 (微软视窗系统)
    + [ ] [macOS] (苹果麦金塔系统)
        + 可能不会为 macOS 版本
    + [X] [Linux]
        + 只提供用于 Archlinux 中 Fcitx5 的包，Fcitx4 和其它发行版请自行打包
+ [Mobile Operating System] (移动平台操作系统)
    + [X] [Android] 4.0+ (谷歌安卓系统)
    + [ ] [iOS]
        + 可能不会为 iOS 版本
